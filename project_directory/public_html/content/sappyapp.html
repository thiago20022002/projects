What is the Sappy Trends?
Sappy Trends is a machine that uses the web to learn data, so far we have created Machine Learning models to accomplish the following tasks :
  • Data Categorization (Supervised)
  • Sentiment Analysis (Supervised)
  • Trending Analysis (Unsupervied Model)
  • Relation of Entities (Unsupervied Model)
